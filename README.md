## Tech Stack

* Python 3.8
* Django 4.0.3

## Run Locally

Clone the project

```bash
  git clone https://github.com/beemensameh/ramadan-series-times.git
```

Go to the project directory

```bash
  cd ramadan-series-times
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  python manage.py runserver 
```
